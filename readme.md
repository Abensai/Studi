## Installation en local

### Pré-requis
* PHP 8.1
* Composer
* Symfony CLI
* Docker-compose
* nodejs et npm

Les pré-requis peuvent être vérifié avec la commande suivante {de la CLI Symfony}: {sauf pour la cas de Docker et Docker-compose}
```bash
symfony check:requirements
```
